Producción por región

Fuentes. Serie mensual nacional: BCRPData (Banco Central de Reserva del Perú), Minería Metálica por mineral — fuente original INEI / MINEM — consultada en vivo vía su API REST (códigos PN01873AM cobre, PN01876AM oro, PN01877AM plata, PN01879AM zinc, etc.; ene-2000→último mes publicado). Detalle por unidad minera: Cuadros Estadísticos Mineros del MINEM (Boletín Estadístico Minero). Participaciones por empresa/región: MINEM/BEM vía prensa (cierre 2025; cobre ene-2026).
Notas. Unidades del BCRP: cobre, zinc, plomo, estaño, hierro y molibdeno en miles de TMF (kt); oro y plata en kg → mostrados en t finas. Si el navegador bloquea la API del BCRP por CORS, el tablero usa puntos mensuales de respaldo (prensa MINEM) y lo indica arriba. Volúmenes por empresa = participación × total (derivado).

Fuentes y actualización

Evolución mensual nacional — BCRP en vivo

El tablero llama a la API de BCRPData al abrir y trae la producción mensual por mineral (2000→último mes). Se actualiza solo cuando el BCRP publica. Si tu red/navegador bloquea la API (CORS), verás puntos de respaldo y un aviso en rojo arriba; en ese caso usa la opción de archivo.

Detalle por unidad minera — MINEM

MINEM › Boletín Estadístico Minero › Cuadros Estadísticos Mineros (Avance Mensual). Selecciona el mineral, pulsa “Cargar por unidad” y elige la hoja de ese metal. Detecta unidad/años/meses y auto-escala. También acepta un data.json multimineral.

{ "minerals": { "Cobre": { "unit":"kt",
   "mines": { "Antamina": {"2025":[...12]}, ... } } } }
