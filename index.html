Actualización automática — “un clic”

Opción A — Manual (ya funciona)

Botón Actualizar (.xlsx) → eliges el archivo recién bajado de Cochilco → todo el tablero se recalcula al último mes. Cero configuración.

Opción B — Automática en GitHub Pages (como tus otras apps)

El tablero ya intenta leer un data.json junto al HTML al abrir. Una GitHub Action baja el Excel de Cochilco cada mes, lo convierte a data.json y lo commitea. Actualizar = automático (o “Run workflow”, un clic). Incluí dos archivos en la entrega:

• actualizar_cochilco.py — descarga el Excel y genera data.json.
• .github/workflows/actualizar_cochilco.yml — corre mensual + manual y commitea.

Súbelos al repo del dashboard. El navegador no puede bajar Cochilco directo (CORS); por eso la descarga la hace la Action en el servidor y el tablero solo lee el JSON same-origin.
